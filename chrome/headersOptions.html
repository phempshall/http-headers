About HTTP Headers

HTTP Headers was written when the original HTTP Headers (HTTP Headers Live) extension sold out to malware injection and adverts. I decided it was time to write my own clean, ad-free, safe version that I didn't have to worry about. This is the outcome, slighly different to the original, a way for me to learn as I developed this version.
